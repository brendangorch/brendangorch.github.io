My Projects

Below, you can find the projects that I have worked on in the past few years. I created these projects by using the concepts I have learned in school alongside the skills I have developed outside of school.

Education Storage

My most recent project, Education Storage, is a JavaFX-based database application created using Apache Derby. The application acts as a "storage" that holds a student's grades throughout their 4 years of being an undergraduate. It also includes a useful GPA calculator that displays yearly GPAs, term GPAs, highest/lowest grades, and total credits. This calculator also allows the user to calculate specific academic averages by selecting the years they want to be included.

Google Web Scraper

Using JavaFX alongside jsoup, I created a simple web scraper during the summer. The application prompts the user to enter keyword(s) for a google search, and the number of pages to search. It then searches through the pages on google, looks at all of the websites within the page range, and checks various locations on each page for the keyword(s) such as title, link, description, and more. Only websites that include the keyword(s) will be returned to the user in a list.

Selection and Merge Sort Simulator

After learning about sorting algorithms in my second-year data structures course, I created a visual simulator for selection sort and merge sort. This was my first time using JavaFX for GUI creation (which I then used for my future projects shown above). The application displays 16 to 128 rectangles, all of different sizes in random order, based on the user's selection. It then visually sorts the rectangles using one of the two sorting algorithms selected by the user.

2D Slime Platformer

My interest in game development lead me to creating a prototype for a 2D platformer. Using Godot's built in scripting lanuage GDScript, I created a very simple and small game where the player controls a slime and jumps through platforms. I implemented a respawn functionality so if the player falls off the platforms, they spawn at the beginning. Currently, this game has no finishing objective, but I may come back to it in the future. I developed this game just to test out the Godot engine and GDScript.
