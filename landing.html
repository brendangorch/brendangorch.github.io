Animal Image Clustering Model

This project aims to integrate unsupervised learning, transfer learning, and K-means clustering for more effective image clustering. The model was able to to group images of 5 unique animal species using K-Means Clustering. The features were extracted from the raw image data for clustering and visualization. The elbow method and silhouette scores were applied to determine optimal cluster count and evaluate performance.

Rustborn

Rustborn is a 2D sandbox game with core mechanics of mining, building, combat, and crafting. Each world is created using procedural content generation techniques such as marching squares and the random-walk algorithm, ensuring that every world is completely unique. Mining and building blocks in the world are dynamically rendered based on the cases of marching square, a feature rarely seen in typical mining mechanics. The entire environment is destructible, giving the player the freedom to shape the world according to their desires.

Education Storage

Education Storage is a JavaFX-based database application created using Apache Derby. The application acts as a "storage" that holds a student's grades throughout their 4 years of being an undergraduate. It also includes a useful GPA calculator that displays yearly GPAs, term GPAs, highest/lowest grades, and total credits. This calculator also allows the user to calculate specific academic averages by selecting the years they want to be included.

Google Web Scraper

Using JavaFX alongside jsoup, I created a simple web scraper application. The application prompts the user to enter keyword(s) for a google search, and the number of pages to search. It then searches through the pages on google, looks at all of the websites within the page range, and checks various locations on each page for the keyword(s) such as title, link, description, and more. Only websites that include the keyword(s) will be returned to the user in a list.
